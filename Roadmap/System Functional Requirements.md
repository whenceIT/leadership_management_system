**System Functional Requirements**



Institutional Average Engine

System must:

•	Calculate rolling averages (monthly, quarterly, annualized)

•	Automatically update benchmarks

•	Adjust for structural changes

•	Maintain historical baseline

Multi-Level Data Drill Architecture

The system must allow drill-down:

Institution

→ Province

→ District/Region

→ Branch

→ Unit

→ Individual Officer

→ Transaction Level



Each level must:

•	Show its contribution to the higher level

•	Show deviation from the institutional average

•	Show deviation from strategic target
